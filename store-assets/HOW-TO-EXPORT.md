# How to Export Store Assets as PNG

## App Icon (512×512)

1. Open `icon-512.html` in Chrome
2. Open DevTools → F12
3. Click the **device toolbar** icon (Ctrl+Shift+M)
4. Set dimensions to **512 × 512**, zoom **100%**
5. Right-click the page → **Capture screenshot**
6. Save as `icon-512.png`

## Feature Graphic (1024×500)

1. Open `feature-graphic-1024x500.html` in Chrome
2. Open DevTools → F12
3. Click the **device toolbar** icon (Ctrl+Shift+M)
4. Set dimensions to **1024 × 500**, zoom **100%**
5. Right-click the page → **Capture screenshot**
6. Save as `feature-graphic-1024x500.png`

## Tips
- Make sure Chrome zoom is 100% (Ctrl+0) before capturing
- Use "Capture screenshot" (not "Capture full size screenshot")
- The icon PNG should have NO rounded corners — Play Store adds them automatically
